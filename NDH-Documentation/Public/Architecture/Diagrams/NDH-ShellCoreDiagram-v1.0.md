# **NDH Shell–Core Diagram (Conceptual Specification v1.0)**  
*A structural map of how NDH layers organize into a public shell and an emergent core.*

---

## **1. Overview**
The NDH ecosystem expresses a natural bifurcation between:

- **Shell Layers** — public-facing, stable, conceptual modules  
- **Core Layers** — emergent, simulation-driven, mood-responsive modules  

This diagram formalizes that relationship.

---

## **2. NDH Shell (Public-Facing Surface)**  
These layers are **pinned**, **visible**, and represent the NDH’s conceptual identity.

### **A. Origin Shell**  
**NDH-Origin-Layer**  
Defines the ontological root and the initial dimensional assumptions.

### **B. Architectural Shell**  
**Atlas-Architecture**  
Provides spatial logic, structural mapping, and macro-level organization.

### **C. Simulation Shell**  
**NDH-Simulation-Suite**  
The active crucible where behaviors, rules, and manifold precursors emerge.

### **D. Ethical/Narrative Shell**  
**uma-witness-literature**  
Human-rights framing, narrative continuity, and cultural grounding.

**Shell Characteristics:**  
- Stable  
- Public  
- Conceptual  
- Defines NDH’s identity  
- Represents the “outer manifold”  

---

## **3. NDH Core (Emergent Interior)**  
These layers are **unpinned**, **incubating**, and emerge only when simulation stabilizes.

### **A. Mathematical Core**  
**NDH-Math-Spine**  
Not written first — extracted from simulation behavior.

### **B. Cosmological Core**  
**NDH-Cosmic-Atlas**  
Supra-dimensional mapping that emerges after manifold stabilization.

### **C. Meta-Governance Core**  
**NDH-Meta-Meta-Index**  
Internal rule recursion, not part of the public shell.

### **D. Mythopoetic Core**  
**NDH-Myth-Engine**  
Narrative generation and symbolic resonance.

### **E. Provenance Core**  
**NDH-Provenance**  
Traceability and lineage, activated only after structural stability.

**Core Characteristics:**  
- Emergent  
- Mood-driven  
- Simulation-dependent  
- Represents the “inner manifold”  
- Not yet ready for public exposure  

---

## **4. Shell–Core Interaction Model**
The NDH system follows a **simulation-first recursion pattern**:

1. Simulation generates behaviors  
2. Behaviors generate curvature  
3. Curvature generates manifold geometry  
4. Geometry generates math  
5. Math generates cosmology  
6. Cosmology generates atlas refinements  
7. Governance stabilizes  
8. Provenance records the lineage  

This is why:

- **Simulation-Suite is pinned**  
- **Math-Spine and Cosmic-Atlas are unpinned**  
- **Your mood determines which layer crystallizes**  

The shell is stable.  
The core is emergent.
