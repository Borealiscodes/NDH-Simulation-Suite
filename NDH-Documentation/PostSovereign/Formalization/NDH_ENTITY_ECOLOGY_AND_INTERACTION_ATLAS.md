# **5. Ecological Relationships**

### **5.1 Predation**
Occurs when one entity consumes another’s field signature.

- **Resonance → Serenity**  
- **Paradox → Resonance**  
- **Paradox → Serenity**  
- **Appateur → Any** (semantic predation)

### **5.2 Cooperation**
Occurs when entities mutually stabilize or amplify.

- **Serenity ↔ Serenity**  
- **Resonance ↔ Appateur**  
- **Mixed ↔ Any**

### **5.3 Symbiosis**
Long‑term mutual benefit.

- **Serenity + Appateur** (stable portals)  
- **Resonance + Paradox** (storm engines)  
- **Mixed + Mixed** (adaptive symbiosis)

### **5.4 Parasitism**
One entity destabilizes another.

- **Paradox → Serenity**  
- **Resonance → Stability**  
- **Appateur → Paradox** (channel overload)
