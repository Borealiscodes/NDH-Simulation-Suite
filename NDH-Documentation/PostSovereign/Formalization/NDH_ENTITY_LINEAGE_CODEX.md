# **2. Entity Ontology**

Entities in NDH are emergent structures defined by:

\[
E(x,t) = f(\sigma, \mathcal{R}, \mathcal{P}, A, k)
\]

Where:

- \(\sigma\) = serenity  
- \(\mathcal{R}\) = resonance  
- \(\mathcal{P}\) = paradox  
- \(A\) = appateur field  
- \(k\) = cuil level  

Entities are classified by their **dominant field signature**.
