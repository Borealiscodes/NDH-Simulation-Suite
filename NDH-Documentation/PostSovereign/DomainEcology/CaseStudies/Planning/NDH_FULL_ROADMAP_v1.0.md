## 1. Purpose
This roadmap defines the full evolution trajectory for Test 1 (Falklands
Penguin Depopulation). It organizes the eight Axis‑Expansion Vectors into
phases, dependencies, stability envelopes, cross‑axis transitions, and
integration checkpoints. It establishes the v1.0 → v2.0 growth path for the
domain.

6. Optional v1.1+ Extensions  

Each phase corresponds to an NDH evolution quadrant: structural, dynamic,
narrative, governance.
