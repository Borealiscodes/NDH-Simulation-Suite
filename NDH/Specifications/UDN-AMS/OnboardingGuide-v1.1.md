# 🌌 **NDH‑AMS Onboarding Guide v1.1**  
### *A step‑by‑step path for learning, using, and navigating the NDH cognitive manifold.*

This guide teaches a new user how NDH‑AMS works, how to move through it, and how to apply it in real situations.

---

# ⭐ 1. **Start With the Big Picture**  
Before touching any axis, a new user needs the “shape” of the system.

- NDH‑AMS is a **9‑axis cognitive manifold**  
- Each axis is a way humans stabilize meaning  
- Axes work together through attachment pairs  
- Axis 05 harmonizes everything  
- Axis 09 and Axis 01 close the loop  
- The triad (06–07–08) stabilizes social meaning
