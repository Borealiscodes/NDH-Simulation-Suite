# ⭐ 3. **Instructor Teaching Techniques**

### **Use Dual‑Framing**  
Every axis should be taught with:

- a **functional framing** (what it does)  
- a **lineage framing** (who embodies it)

This reinforces the dual‑label structure.

### **Use Progressive Complexity**  
Start with:

1. plain language  
2. conceptual framing  
3. structural framing  
4. operator framing  
5. basin framing  
6. threshold framing  

Never start with math.

### **Use Attachment‑First Integration**  
When teaching cross‑axis behavior, always begin with attachment pairs before transitions.

### **Use Loop‑Closure Anchoring**  
Return to Axis 01 ↔ Axis 09 frequently — it stabilizes the entire learning experience.

# ⭐ 4. **Instructor Safety Guidelines**

### **Avoid identity interpretation**  
Axis 02 is about *stabilization*, not *analysis* of identity.

### **Avoid moral adjudication**  
Axis 06 clarifies responsibility; it does not assign blame.

### **Avoid therapeutic framing**  
Axis 04 and Axis 07 involve compassion, not therapy.

### **Avoid historical adjudication**  
Axis 08 preserves memory; it does not rewrite it.

### **Avoid metaphysical claims**  
Axis 09 clarifies awareness; it does not prescribe beliefs.

# ⭐ 5. **Exercises for Learners**

### **Exercise 1 — Axis Identification**  
Give learners short scenarios and ask which axis activates.

Jump deeper: **Quick Reference Card**

### **Exercise 2 — Attachment Mapping**  
Give learners two axes and ask how they reinforce each other.

### **Exercise 3 — Threshold Spotting**  
Give learners ambiguous situations and ask what threshold is triggered.

### **Exercise 4 — Triad Analysis**  
Give learners a societal scenario and ask how 06–07–08 interact.

### **Exercise 5 — Loop‑Closure Walkthrough**  
Ask learners to explain how awareness stabilizes reality and vice versa.

### **Exercise 6 — Emergent Case Reconstruction**  
Give learners a multi‑axis scenario and ask them to map transitions.

# ⭐ 6. **Instructor Evaluation Methods**

### **Check for Axis Confusion**  
Learners should be able to distinguish:

- identity vs story  
- personal vs structural compassion  
- memory vs narrative  
- awareness vs reality  

### **Check for Attachment Understanding**  
Learners should know the four pairs + Axis 05’s universal role.

### **Check for Threshold Recognition**  
Learners should identify triggers without prompting.

### **Check for Transition Logic**  
Learners should understand why 02 → 06 or 03 → 08 occurs.

### **Check for Emergent Stability**  
Learners should be able to explain how the manifold resolves complexity.
