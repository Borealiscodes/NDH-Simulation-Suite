# ⭐ 1. **What NDH‑AMS *Is***  
NDH‑AMS is a **9‑axis cognitive map** that explains how humans stabilize:

- truth  
- identity  
- story  
- compassion  
- ethics  
- memory  
- awareness  

Each axis is a *perspective* and a *skill*, grounded in a lineage anchor.
