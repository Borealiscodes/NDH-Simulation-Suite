## ⭐ 1. Basis: surfaces in the NDH manifold

A **transition surface** is a geometric subset of the NDH manifold:

\[
\Sigma_k = \{ x \in \mathcal{M} \mid R_k(x) = \Theta_k \}
\]

where:

- \(\mathcal{M}\) = NDH manifold  
- \(R_k(x)\) = readiness function for FO‑I, FO‑II, FO‑III  
- \(\Theta_k\) = threshold for FO‑class \(k\)

Each \(\Sigma_k\) is a **codimension‑1 surface** separating regions where FO transitions are inactive vs active.
