### NDH Stability Simulator v2.0  
*Interactive sandbox for NDH basins, tensors, and ladder modes.*

---

## 1. Purpose  

The NDH Stability Simulator v2.0 is a **standalone interactive environment** for:

- exploring basins, corridors, margins, and K‑space  
- visualizing tensor fields and stability envelopes  
- testing ladder mode transitions and accessibility profiles  
- validating HRD rules and sensory behavior before deployment  

It is the **safe playground** for NDH physics.

---

## 2. Core capabilities  

- **Geometry exploration:**  
  - visualize basins, corridors, margins, K‑space layers  
  - inspect geometry envelopes over time  

- **Tensor field visualization:**  
  - curvature, stress, resonance, fold, invariants  
  - stability inequalities and threshold crossings  

- **Ladder mode testing:**  
  - Normal → Calm → Serenity → Tranquility → Peace  
  - Dark Mode overlays and transitions  

- **Accessibility scenario testing:**  
  - audioOnly, hapticPrimary, visualEase, cognitiveEase  
  - sensoryMode: minimal/normal/intense (HRD‑bounded)  

- **HRD rule validation:**  
  - tensor spikes, sensory intensity, margin proximity  
  - Dark Mode visibility and multi‑manifold safety  

---

## 3. Simulator modes  

- **Single‑manifold mode:**  
  - focus on one manifold’s basins, tensors, and ladder behavior  

- **Multi‑manifold mode:**  
  - visualize multiple manifolds in parallel  
  - show inter‑manifold bridges and cross‑manifold effects  

- **Accessibility test mode:**  
  - run scripted scenarios across profiles  
  - record JDTR logs for review  

- **Developer debug mode:**  
  - expose raw tensors, geometry, HRD decisions, JDTR events  

---

## 4. Integration with NDH Runtime  

- connects to **NDH Runtime Engine v2.0** via API  
- can run in:  
  - **live mode:** attached to a running NDH instance  
  - **offline mode:** replay JDTR logs and scenarios  
- uses NDH Sensory Engine v2.0 for sensory previews  
- respects HRD and accessibility constraints at all times  

---

## 5. Outputs  

- **visualizations:** geometry, tensors, ladder modes, manifolds  
- **reports:** stability envelopes, HRD violations, accessibility impacts  
- **JDTR logs:** full trace of simulated runs  
- **exportable scenarios:** reusable test cases for regression
