## 🧭 **Versioning Rules**
- All diagrams use semantic versioning: `v1.0`, `v1.1`, etc.  
- Updates must be additive, not destructive.  
- Deprecated diagrams move to:  
  ```
  NDH/Architecture/Book/assets/archive/
  ```

## 🧱 **Structural Requirements**
- All PNGs follow the NDH global spec:  
  2400×2400 px, Inter font family, NDH palette, no gradients, no transparency.  
- All filenames must begin with `ndh-` and end with `-vX.Y.png`.  
- No inline images inside Markdown — only PNG references.
